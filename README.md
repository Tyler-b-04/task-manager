# Task Manager App

## Project Description

This project is a customized task management web application built for my Business Application Development course. The app allows users to organize tasks in a simple interface while practicing front-end development concepts such as components, styling, file organization, and user interaction.

I customized the project with a dark gray and orange theme to make the design more personal and professional. This project helped me practice building a more complete application instead of only writing small JavaScript examples.

## Technologies Used

- Next.js
- JavaScript
- TypeScript
- CSS
- Node.js
- npm
- Git and GitHub
